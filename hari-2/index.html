📖 Konsep

Apa itu CSS?

🎨

Tampilan Visual

CSS mengontrol warna, font, ukuran, jarak, animasi — semua yang terlihat di mata Anda pada sebuah website.

📐

Layout & Posisi

CSS menentukan di mana setiap elemen berada — kiri, kanan, tengah, kolom, baris — tanpa mengubah HTML.

⚡

Efek & Animasi

Hover effects, transisi halus, animasi bergerak — semuanya bisa dilakukan CSS tanpa JavaScript.

/* Sintaks CSS: Selector { Properti: Nilai; } */

}

.kartu {

  border-radius: 16px;

}
🔌 Cara Menulis

3 Cara Menulis CSS

✅ External CSS

File style.css terpisah. Cara terbaik dan paling profesional — reusable di banyak halaman.

<!-- Di dalam <head> -->
<link rel="stylesheet"
      href="style.css">

⚠️ Internal CSS

Di dalam tag <style> di bagian head. Cocok untuk satu halaman, tapi tidak reusable.

<style>
  h1 { color: blue; }
  p  { font-size: 16px; }
</style>

❌ Inline CSS

Langsung di atribut style. Hindari! Sulit di-maintain dan tidak bisa direuse.

<p style="color: red;
       font-size: 16px;
       font-weight: bold">
  <!-- Hindari cara ini! -->
</p>

📦 Box Model

Memahami CSS Box Model

margin
border
padding
content
width: 300px; height: 200px;

Content — area konten utama. width/height mengatur ini.

padding: 20px 16px;

Padding — ruang di dalam border. Tidak memisahkan dari elemen lain.

border: 2px solid #3b82f6;

Border — garis tepi. Bisa solid, dashed, dotted, double, dll.

margin: 24px auto;

Margin — jarak ke elemen lain. auto = pusatkan horizontal.
