🔗 Tugas 1 — Setup CSS

⭐ Mudah

Buat file style.css dan hubungkan ke semua halaman HTML Anda.

Buat file baru bernama style.css di folder yang sama dengan index.html
Tambahkan <link rel="stylesheet" href="style.css"> di dalam <head> setiap file HTML
Reset default browser: hilangkan margin/padding bawaan dengan * { box-sizing: border-box; margin: 0; padding: 0; }
Import Google Fonts di CSS: gunakan font Inter atau Poppins
/* style.css — Mulai dari sini */

/* 1. Import font dari Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap');

/* 2. Reset global */
*, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }

/* 3. Base typography */
body {
  font-family: 'Inter', sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #1e293b;
  background-color: #f8fafc;
}

🎨 Tugas 2 — Styling Header & Nav

⭐ Mudah

Buat header dan navigasi yang terlihat profesional menggunakan Flexbox.

A

Styling Header

Beri header: background warna gelap atau gradient, padding 20px, warna teks putih, lebar penuh (width: 100%). Nama Anda di h1 harus menonjol.

backgroundpaddingcolorlinear-gradient
B

Navbar Horizontal dengan Flexbox

Buat nav menjadi horizontal menggunakan display:flex. Link nav tidak bergaris bawah, berwarna putih/terang, dan berubah saat di-hover (opacity atau warna berbeda).

display: flexgapa:hovertext-decoration
header {
  background: linear-gradient(135deg, #1e3a5f, #2563eb);
  color: white;
  padding: 20px 40px;
  display: flex;
  justify-content: space-between;
  align-items: center;
}
nav a { color: white; text-decoration: none; padding: 8px 16px; border-radius: 6px; transition: all 0.3s; }
nav a:hover { background: rgba(255,255,255,0.2); }

🎭 Tugas 4 — Styling Form Kontak

⭐⭐ Menengah

Buat form kontak.html tampil rapi dan profesional dengan CSS.

Input dan textarea: padding 12px, border 1px solid #e2e8f0, border-radius 8px, lebar penuh (width: 100%)
Input saat focus: border berubah warna biru + box-shadow biru transparan (gunakan :focus)
Tombol submit: background biru, teks putih, padding, border-radius, hover effect
Label: display block, font-weight bold, margin-bottom 6px
Grup field: margin-bottom 20px untuk jarak antar field
input, textarea, select {
  width: 100%; padding: 12px 16px;
  border: 1px solid #e2e8f0; border-radius: 8px;
  font-size: 15px; transition: border-color .3s, box-shadow .3s;
  outline: none;
}
input:focus, textarea:focus {
  border-color: #3b82f6;
  box-shadow: 0 0 0 3px rgba(59,130,246,0.15);
}

🚀 Tugas 5 — Tantangan: Dark Mode

⭐⭐⭐ Tantangan

Tambahkan CSS Variables dan buat tema gelap (dark mode) untuk website Anda.

✦

Gunakan CSS Custom Properties (Variables)

Deklarasikan variabel warna di :root, lalu pakai var(--nama-variable) di seluruh CSS Anda. Buat versi light dan dark theme. Toggle dengan JavaScript saat tombol diklik.

:root { --var }var(--warna)[data-theme="dark"]
:root { /* Light theme */
  --bg: #f8fafc; --text: #1e293b;
  --card-bg: white; --accent: #3b82f6;
}
[data-theme="dark"] { /* Dark theme */
  --bg: #0f172a; --text: #e2e8f0;
  --card-bg: #1e293b;
}
body { background: var(--bg); color: var(--text); }

💡 Tips & Panduan

  • Tombol toggle: <button onclick="document.documentElement.dataset.theme = 'dark'">Dark</button>
  • Atau lebih cerdas: toggle antara light dan dark dengan if/else di JavaScript
  • Simpan preferensi di localStorage agar tetap tersimpan saat halaman di-refresh
  • Gunakan localStorage.setItem('theme', 'dark') dan getItem('theme') saat halaman dimuat
