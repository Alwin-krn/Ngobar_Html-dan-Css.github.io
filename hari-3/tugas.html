📐 Tugas 1 — CSS Grid Layout

⭐ Mudah

Ubah layout halaman profil Anda dari Flexbox ke CSS Grid yang lebih kompleks.

A

Grid Layout Halaman Utama

Buat layout 2 kolom untuk section "Tentang Saya": kolom kiri untuk foto (1fr), kolom kanan untuk teks bio (2fr). Gunakan CSS Grid.

display: gridgrid-template-columns: 1fr 2frgap
B

Galeri Keahlian dengan Grid

Ubah daftar keahlian menjadi galeri card dengan CSS Grid. Gunakan repeat(auto-fit, minmax(200px, 1fr)) agar responsif otomatis.

auto-fitminmax()responsive otomatis
.about-section {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 40px;
  align-items: start;
}

.skills-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 16px;
}

✨ Tugas 2 — Animasi & Transisi

⭐⭐ Menengah

Tambahkan kehidupan pada website Anda dengan animasi dan transisi CSS!

1

Animasi Loading (Fade-in)

Buat seluruh konten body muncul dengan fade-in saat halaman pertama dibuka. Gunakan @keyframes dan animation pada body atau .container.

@keyframes fadeinopacity: 0 → 1animation-fill-mode: both
2

Hover Card dengan Efek Lift

Card keahlian saat di-hover: naik ke atas (translateY(-6px)), shadow lebih besar, border berubah warna. Semua transisi dalam 0.3s ease.

transform: translateY()box-shadowtransition: all .3s ease
3

Animasi Progress Bar

Buat progress bar untuk menampilkan persentase keahlian (HTML: 85%, CSS: 80%, dll). Animasikan lebar bar dari 0% ke target saat halaman dimuat.

@keyframes growwidth: 0 → 85%animation-delay
/* Fade-in masuk halaman */
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
.card { animation: fadeInUp 0.6s ease both; }
.card:nth-child(2) { animation-delay: 0.1s; }
.card:nth-child(3) { animation-delay: 0.2s; }

📱 Tugas 3 — Responsive Design

⭐⭐ Menengah

Buat website Anda tampil sempurna di HP, tablet, dan desktop.

Desktop (> 1024px): layout 3 kolom, padding besar, font besar
Tablet (≤ 768px): navbar jadi hamburger menu (hanya ikon ☰), kolom jadi 2
HP (≤ 480px): semua kolom jadi 1, padding mengecil, h1 lebih kecil
Gambar profil: di desktop di samping teks, di HP pindah ke atas (column)
Uji di Chrome DevTools: tekan F12 → ikon HP di atas untuk simulasi mobile
/* HP kecil */
@media (max-width: 480px) {
  .about-section {
    grid-template-columns: 1fr; /* 1 kolom */
    text-align: center;
  }
  .foto-profil {
    width: 120px; margin: 0 auto 20px;
  }
  h1 { font-size: clamp(22px, 5vw, 36px); }
}

🎛️ Tugas 4 — CSS Variables + Dark Mode

⭐⭐ Menengah

Implementasikan CSS Variables dan tombol Dark Mode yang fungsional.

✓

Refactor CSS dengan Variables

Ganti semua warna hardcoded di CSS Anda dengan CSS Variables di :root. Minimal: --bg, --text, --primary, --card, --border, --shadow.

:root { }var(--nama)
✓

Tombol Toggle Dark/Light Mode

Tambahkan tombol 🌙/☀️ di navbar. Saat diklik, toggle class atau data-theme di <html>. Simpan preferensi di localStorage agar tidak reset saat refresh.

[data-theme="dark"]localStoragetoggle class
/* JavaScript untuk toggle */
const btn = document.getElementById('theme-btn');
const html = document.documentElement;

// Load preferensi tersimpan
if (localStorage.getItem('theme') === 'dark') {
  html.dataset.theme = 'dark';
}
btn.addEventListener('click', () => {
  const isDark = html.dataset.theme === 'dark';
  html.dataset.theme = isDark ? 'light' : 'dark';
  localStorage.setItem('theme', html.dataset.theme);
});
