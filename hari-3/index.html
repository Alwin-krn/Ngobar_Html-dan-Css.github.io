📐 CSS Grid

Layout 2D dengan CSS Grid

.container {
  display:                grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows:    auto;
  gap:                   20px;
}

/* Item bentang 2 kolom */
.featured {
  grid-column: span 2;
}

/* Posisi eksplisit */
.hero {
  grid-column: 1 / 4;

⚖️ Grid vs Flexbox

Kapan Pakai Masing-Masing?

📏 Flexbox — 1 Dimensi

Terbaik untuk baris ATAU kolom saja.

  • ✅ Navbar / menu horizontal
  • ✅ Tombol dengan ikon berjejer
  • ✅ Centering vertikal + horizontal
  • ✅ Card dalam satu baris
  • ✅ Header: logo kiri, menu kanan
display:      flex;
align-items: center;
gap:         16px;

📐 CSS Grid — 2 Dimensi

Terbaik untuk baris DAN kolom sekaligus.

  • ✅ Layout halaman penuh
  • ✅ Dashboard dengan panel
  • ✅ Galeri foto / produk
  • ✅ Artikel dengan sidebar
  • ✅ Masonry / Pinterest layout
display:                grid;
grid-template-columns: 1fr 3fr;
gap:                   24px;
✨ Animasi

Animasi & Transisi CSS

Transisi (Hover Effect)

.tombol {
  background:  #8b5cf6;
  transition:  all 0.3s ease;
}

.tombol:hover {
  background:  #7c3aed;
  transform:   translateY(-3px);
  box-shadow:  0 8px 24px rgba(139,92,246,.4);
}

Keyframe Animation

@keyframes naik-turun {
  0%   { transform: translateY(0px);   }
  50%  { transform: translateY(-20px); }
  100% { transform: translateY(0px);   }
}

.elemen {
  animation: naik-turun 2s ease infinite;
}

🎛️ CSS Variables

CSS Custom Properties

/* Deklarasi di :root = variabel global */
:root {
  --warna-primer: #8b5cf6;
  --warna-teks:   #1e293b;
  --rounded:      12px;
  --shadow:       0 4px 20px rgba(0,0,0,.08);
  --font:         'Inter', sans-serif;
}

/* Pakai dengan var() */
.tombol {
  background:    var(--warna-primer);
  border-radius: var(--rounded);
}

/* Dark mode: ganti satu blok! */
[data-theme="dark"] {
  --warna-teks: #e2e8f0;
  --bg:         #0f172a;
}

✅ Keunggulan CSS Variables

🎨 Konsistensi — ubah satu, berubah semua
🌙 Dark Mode mudah tanpa menulis ulang
🔁 DRY — Don't Repeat Yourself
⚡ Dinamis — bisa diubah via JavaScript

Ubah variabel via JavaScript

// Ubah warna tema real-time
document.documentElement
  .style.setProperty(
    '--warna-primer', '#ef4444'
  );
📱 Responsive

Responsive Design & Media Query

/* Desktop (default) */
.container {
  display:                grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap:                   24px;
}

/* Tablet — max 768px */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr 1fr;
  }
  nav {
    flex-direction: column;
  }
}

/* HP — max 480px */
@media (max-width: 480px) {
  .container {
    grid-template-columns: 1fr;
  }
  h1 { font-size: 24px; }
}
