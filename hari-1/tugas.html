🏗️ Tugas 1 — Struktur Dasar HTML

⭐ Mudah

Buat file HTML pertama dari nol. Pelajari contoh di bawah, lalu buat versi Anda sendiri.

📄 Contoh Kode — index.html CONTOH LENGKAP
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Budi Santoso – Web Developer</title>
</head>
<body>

  <header>
    <h1>Halo! Saya Budi Santoso</h1>
    <p>Web Developer · Jakarta, Indonesia</p>
    <nav>
      <a href="#tentang">Tentang</a> |
      <a href="#keahlian">Keahlian</a> |
      <a href="#kontak">Kontak</a>
    </nav>
  </header>

  <main>
    <section id="tentang">
      <h2>Tentang Saya</h2>
      <img src="foto.jpg" alt="Foto Budi Santoso" width="150">
      <p>Saya seorang web developer dengan pengalaman 3 tahun</p>
      <p>Saya menyukai desain UI/UX dan coding frontend.</p>
    </section>

    <section id="keahlian">
      <h2>Keahlian Saya</h2>
      <ul>
        <li>HTML5 – Struktur web</li>
        <li>CSS3 – Tampilan web</li>
        <li>JavaScript – Interaktivitas</li>
        <li>Git – Version control</li>
      </ul>
    </section>
  </main>

  <footer>
    <p>&copy; 2025 Budi Santoso. Dibuat dengan ❤️</p>
  </footer>

</body>
</html>

📋 Tugas 2 — Form Kontak HTML

⭐ Mudah

Buat halaman kontak.html dengan form lengkap. Pelajari contoh berikut.

📄 Contoh Kode — kontak.html CONTOH LENGKAP
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Kontak Saya</title>
</head>
<body>
  <h1>Hubungi Saya</h1>
  <form action="#" method="POST">

    <!-- Nama lengkap (wajib diisi) -->
    <label for="nama">Nama Lengkap *</label>
    <input type="text" id="nama" name="nama"
      placeholder="Masukkan nama lengkap" required>

    <!-- Email (validasi otomatis) -->
    <label for="email">Email *</label>
    <input type="email" id="email" name="email"
      placeholder="nama@email.com" required>

    <!-- Nomor HP -->
    <label for="hp">Nomor HP</label>
    <input type="tel" id="hp" name="hp" placeholder="08xxxxxxxxxx">

    <!-- Subjek pilihan -->
    <label for="subjek">Subjek</label>
    <select id="subjek" name="subjek">
      <option value="">-- Pilih Subjek --</option>
      <option value="proyek">Kerja Sama Proyek</option>
      <option value="konsultasi">Konsultasi</option>
      <option value="lainnya">Lainnya</option>
    </select>

    <!-- Pesan (textarea) -->
    <label for="pesan">Pesan *</label>
    <textarea id="pesan" name="pesan" rows="5"
      placeholder="Tulis pesan Anda di sini..." required></textarea>

    <!-- Persetujuan -->
    <input type="checkbox" id="setuju" name="setuju" required>
    <label for="setuju">Saya setuju dengan syarat & ketentuan</label>

    <button type="submit">Kirim Pesan</button>
  </form>
</body>
</html>

🔗 Tugas 4 — Link & Gambar

⭐⭐ Menengah

Kuasai cara menggunakan tag <a> dan <img> dengan benar.

📄 Contoh Kode — Link & Gambar CONTOH LENGKAP
<!-- LINK INTERNAL (ke halaman lain) -->
<a href="kontak.html">Pergi ke Kontak</a>

<!-- LINK EKSTERNAL (buka tab baru) -->
<a href="https://github.com" target="_blank" rel="noopener">
  GitHub Saya
</a>

<!-- LINK ANCHOR (lompat ke section) -->
<a href="#tentang">Gulir ke Tentang</a>

<!-- LINK EMAIL -->
<a href="mailto:budi@email.com">Kirim Email</a>

<!-- GAMBAR BIASA -->
<img src="foto.jpg" alt="Foto Profil Budi" width="200" height="200">

<!-- GAMBAR DENGAN PLACEHOLDER -->
<img src="https://placehold.co/200x200/3b82f6/fff?text=Foto"
     alt="Foto placeholder" width="200">

<!-- GAMBAR DENGAN LINK (klik gambar = buka link) -->
<a href="https://github.com" target="_blank">
  <img src="github-logo.png" alt="GitHub" width="40">
</a>
