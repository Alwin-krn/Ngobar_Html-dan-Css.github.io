📖 Konsep Dasar

Apa itu HTML?

🏗️

Kerangka Website

HTML adalah "tulang" website. Tanpa HTML, tidak ada yang bisa ditampilkan. CSS = tampilan, JS = fungsi.

📝

Markup Language

Bukan bahasa pemrograman! HTML "menandai" konten agar browser tahu cara menampilkannya.

🌐

Standar Web Global

Semua website pakai HTML. Versi terbaru adalah HTML5 (2014) dengan fitur-fitur modern yang kaya.

<!-- HTML memberi MAKNA pada konten -->

<h1>Judul Utama Halaman</h1>    <!-- Ini judul, bukan sekadar teks besar -->
<p>Ini paragraf artikel saya.</p>    <!-- Ini paragraf, bukan sekadar teks biasa -->
<button>Klik Saya</button>          <!-- Ini tombol yang bisa diklik -->
🏛️ Struktur

Struktur Dokumen HTML

<!DOCTYPE html>        <!-- Deklarasi HTML5 -->
<html lang="id">

  <head>               <!-- Informasi halaman -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width">
    <title>Judul di Tab Browser</title>
  </head>

  <body>               <!-- Konten yang terlihat -->
    <h1>Halo Dunia!</h1>
    <p>Website pertama saya.</p>
  </body>

</html>
<!DOCTYPE>
Memberitahu browser: ini HTML5
<html>
Pembungkus seluruh dokumen
<head>
Meta info: charset, title, link CSS
<meta charset>
Encoding karakter (wajib ada!)
<meta viewport>
Agar tampil bagus di HP
<title>
Teks di tab/judul browser
<body>
Semua konten yang dilihat user

📋 Form

Membuat Form HTML

<form action="proses.php" method="POST">

  <!-- Input teks -->
  <label for="nama">Nama Lengkap:</label>
  <input type="text" id="nama" name="nama" required>

  <!-- Input email -->
  <label for="email">Email:</label>
  <input type="email" id="email" placeholder="nama@email.com">

  <!-- Textarea -->
  <textarea name="pesan" rows="4"></textarea>

  <!-- Tombol Submit -->
  <button type="submit">Kirim Pesan</button>

</form>

radioPilih satu opsi

datePilih tanggal
✅ Semantic HTML5

Semantic HTML vs Non-Semantic

❌ Non-Semantic (Jangan!)
<div id="header">...</div>
<div id="nav">...</div>
<div id="main">
  <div id="article">...</div>
  <div id="sidebar">...</div>
</div>
<div id="footer">...</div>

Browser & mesin pencari tidak tahu makna setiap <div>

✅ Semantic HTML5 (Gunakan!)
<header>...</header>
<nav>...</nav>
<main>
  <article>...</article>
  <aside>...</aside>
</main>
<footer>...</footer>

Jelas maknanya — bagus untuk SEO, aksesibilitas, dan tim

<header>

Kepala halaman

<section>

Bagian konten

<article>

Konten mandiri

<figure>

Gambar + caption

💻 Contoh Lengkap

Halaman Profil HTML Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Profil Saya</title>
</head>
<body>

  <header>
    <h1>Budi Santoso — Web Developer</h1>
    <nav>
      <a href="#tentang">Tentang</a> |
      <a href="#keahlian">Keahlian</a> |
      <a href="#kontak">Kontak</a>
    </nav>
  </header>

  <main>
    <section id="tentang">
      <h2>Tentang Saya</h2>
      <p>Saya seorang web developer dari Jakarta.</p>
    </section>
    <section id="keahlian">
      <h2>Keahlian</h2>
      <ul>
        <li>HTML5</li>
        <li>CSS3</li>
        <li>JavaScript</li>
      </ul>
    </section>
  </main>

  <footer>
    <p>&copy; 2025 Budi Santoso</p>
  </footer>

</body>
</html>
